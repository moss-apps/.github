Welcome to **Moss Apps**. We are dedicated to building high-performance, privacy-conscious applications designed to deliver uncompromised user experiences. 

Our ecosystem spans from audiophile-grade media playback to secure digital storage, built with a strict focus on robust architecture, modern tooling, and cross-platform compatibility.

### Our Philosophy
* **Uncompromised Performance:** We bypass standard system limitations when necessary, utilizing system-level languages to achieve the best possible results.
* **Privacy by Default:** Your data and media belong to you. We build tools that isolate and protect user information from prying eyes.
* **Modern Architecture:** We leverage the best contemporary frameworks to ensure our applications are fast, responsive, and maintainable.

---

### Featured Projects

#### Flick Player
A high-fidelity music player for Android engineered for audio enthusiasts. Flick bypasses the standard Android audio stack, utilizing a custom-built, Rust-based UAC 2.0 implementation to deliver bit-perfect audio directly to external USB DACs and amplifiers. Features include gapless playback, automated local library scanning, and a 10-band equalizer.

#### Locker
A secure, zero-compromise media vault. Locker provides a simple, intuitive interface to encrypt and isolate your personal images, videos, and documents behind a password-protected gateway, ensuring your private media remains strictly confidential.

#### Moss Web
The central hub for the Moss ecosystem. Built for speed and responsiveness, it serves as the landing page and informational center for all our applications.
